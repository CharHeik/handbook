# Data Analysis
Data analysis performed by OET revolves around gathering the data required for modeling the power systems, from various aspects.

- It is both the technical and market data, utilized as a backbone for the power system expansion, unit commitment/economic dispatch, and the power flow calculation.
- Data analysis plays a crucial role, prior the computer simulation, as the correct input data is the foundation for all the other work done by OET.
